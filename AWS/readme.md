## 🌐 1. What is Cloud Computing?

Cloud Computing is the on-demand delivery of IT resources over the internet with pay-as-you-go pricing.

👉 Resources include:
- Servers  
- Storage  
- Databases  
- Networking  
- Software  

---

## 💰 2. Cloud Pricing Model

👉 Cloud follows a **rent-based model**

- Pay only for what you use  
- No upfront hardware cost  
- Scalable billing  

---

## 🏢 3. Cloud Service Providers

Popular cloud providers:

- Amazon Web Services (AWS)  
- Microsoft Azure  
- Google Cloud Platform (GCP)  

---

## 📖 4. Real-Life Use Case

### 🛒 E-Commerce Example

Imagine Raju builds an online store.

### ❌ Problems without cloud:
- Server downtime  
- Security risks  
- No backups  
- Cannot handle high traffic  

### ✅ With Cloud:
- 24/7 availability  
- Automatic scaling  
- Secure environment  
- Backup & recovery

## ☁️ 5. Types of Cloud Services (Service Models)

<div align="center">

<table>
<tr>
<td align="center">

### 🏗️ IaaS  
Infrastructure as a Service  

🧠 Concept:  
You get raw infrastructure  

🏪 Example:  
Empty shop (you build everything)

</td>

<td align="center">

### 🍦 PaaS  
Platform as a Service  

🧠 Concept:  
Platform ready, just deploy code  

🏪 Example:  
Furnished ice cream shop

</td>

<td align="center">

### 🧁 SaaS  
Software as a Service  

🧠 Concept:  
Ready-to-use software  

🏪 Example:  
Ice cream vending machine

</td>
</tr>
</table>

</div>
