# 🌐 1. Cloud Computing

## 🔹 What is Cloud Computing?

Cloud Computing is the on-demand delivery of IT resources over the internet with pay-as-you-go pricing.

👉 Resources include:
- Servers  
- Storage  
- Databases  
- Networking  
- Software  

---

## 💰 Pricing Model

- Pay only for what you use  
- No upfront cost  
- Scalable billing  

---

## 🏢 Cloud Providers

- AWS  
- Microsoft Azure  
- Google Cloud  

---

## 📖 Real-Life Use Case

### 🛒 E-Commerce Example

❌ Without Cloud:
- Downtime  
- No scalability  
- Security issues  

✅ With Cloud:
- 24/7 uptime  
- Auto scaling  
- Secure + backup

# ☁️ 2. AWS (Amazon Web Services)

## 🔹 What is AWS?

AWS is a cloud computing platform launched in **2006** that provides scalable IT resources.

---

## 📈 Evolution

- Started with EC2 & S3  
- Now 200+ services  
- Includes AI, ML, IoT  

---

## 🌍 Real-World Usage

Used by:
- Netflix  
- NASA  
- Airbnb
